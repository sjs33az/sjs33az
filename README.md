### 🚀 Projects

#### 1. 사용자 맞춤형 AI 정서 케어 시스템 - Heartbeats
*사용자의 정서적 안정을 위한 실시간 맞춤형 상담 서비스*
* **주요 역할**: 백엔드 시스템(FastAPI), 상담 세션 관리 API, 프론트엔드 UI 및 시연 설계
* **핵심 성과**: 
    * 맥락 유지를 위한 `Prompt Agent` 설계로 과거 상담 내용 연속성 확보
    * `3-DB(General, Sensitive, Audit)` 물리적 격리 구조 및 `AES-256-GCM` 본문 암호화 적용으로 보안 강화
    * `Function Calling` 기반 실시간 위험 감지 및 위기 상황 대응 로직 구현

#### 2. 나만의 취향 맛집 분석기 - ME슐랭
*리뷰 과부하를 해결하고 개인화된 맛집 추천을 제공하는 AI 서비스*
* **주요 역할**: Flask 기반 백엔드 및 프론트엔드 구현, 크롤링 및 토큰화, 시연 설계
* **핵심 성과**:
    * `BERTopic`을 활용한 리뷰 토픽 모델링 및 카테고리별 정서 점수화
    * `RoBERTa` 기반 Zero-shot 감성 분석 모델 적용으로 사용자 의도 파악 정확도 향상
    * 카테고리 가중치 설정 인터페이스 구현을 통해 사용자 중심의 추천 로직 제공

#### 3. 병원 폐업 구조적 요인 분석
*재무 데이터와 지역 특성을 결합한 데이터 분석 프로젝트*
* **주요 역할**: 데이터셋 확보, 지역 특성 분석, DB 연결 및 시각화
* **핵심 성과**:
    * `재무 데이터`에 `지역 파생 변수`를 통합하여 폐업 예측 모델 성능 개선 (`AUC 76% → 79%`, `Recall 31% → 46%`)
    * `Random Forest` 모델 활용 모델 영향 피처 순위 산출 및 지역 의료 생태계 관점의 시사점 도출
